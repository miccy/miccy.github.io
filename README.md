## 🚀 O mně

Jsem nezávislý full-stack vývojář s patnáctiletou zkušeností, specializující se na vývoj mobilních aplikací. V současné době se zaměřuji na web3 aplikace a zkoumám možnosti integrace AI a vývoj vlastních LLM. Mým cílem je vytvářet robustní, uživatelsky přívětivá řešení, která propojují tradiční a nově vznikající technologie. Kladu důraz hlavně na security a automatizaci procesů, jak při vývoji, tak pro zákazníky a uživatele.
